# Aircraft_Battle_Bata

基于 [SDL](https://www.libsdl.org/) 开发的小游戏 《太空飞机大战》 未完成

## 文件夹和文件介绍

`Game`      游戏源代码

`bin`       游戏可执行文件，和必要的运行库（游戏的安装包我会发布在 Release）

`icon`      游戏图标

`img`       游戏素材文件

`Compile`   编译源代码的 `.bat` 文件

`SDL`       所用到的图形库
